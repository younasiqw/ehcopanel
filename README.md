# Install
```
wget -O setup_ehco_web.sh https://raw.githubusercontent.com/younasiqw/ehcopanel/refs/heads/main/setup_ehco_web.sh && chmod +x setup_ehco_web.sh && sudo ./setup_ehco_web.sh
```
# Uninstall
```
wget -O uninstall_ehco_web.sh https://github.com/younasiqw/ehcopanel/blob/main/uninstall_ehco_web.sh && chmod +x uninstall_ehco_web.sh && sudo ./uninstall_ehco_web.sh
```
